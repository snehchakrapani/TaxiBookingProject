# TaxiBookingProject

TaxiBookingProject is a full-stack cab booking application with:

- `TaxiBookingService`: ASP.NET Core backend API
- `taxi-booking-frontend`: React + Vite frontend

## Main Features

- Rider and driver authentication
- Trip booking with pickup and drop locations
- Fare and ETA calculation
- Driver ride flow and ride status updates
- Profile and wallet-related UI improvements

## Tech Stack

- Frontend: React, Vite, Redux Toolkit, RTK Query, Material UI, Leaflet
- Backend: ASP.NET Core, Entity Framework Core, SQL Server, JWT authentication

## Project Structure

- `TaxiBookingService/` - backend API and database logic
- `taxi-booking-frontend/` - frontend application

## Notes

- Generated build output and verification folders are ignored from Git.
- Environment-specific values should be configured locally before running the project.
